## ✨ ما الجديد في الإصدار 2.1؟

### 🎯 الميزات الجديدة

- **⭐ نظام المفضلة المتقدم** – إضافة الإذاعات إلى المفضلة، ورفعها تلقائياً إلى أعلى القائمة بنفس ترتيب الإضافة (آخر اختيار أولاً). زر قلب أحمر مميز.
- **👆 التحكم بالإصبع (Swipe Gestures)** – التمرير في الجزء العلوي من الشاشة → تغيير الإذاعة، التمرير في الجزء السفلي → تغيير مستوى الصوت.
- **🔘 إعادة تصميم أزرار الإذاعات** – زر تشغيل وزر مفضلة ظاهران دائماً، مع زر قائمة ☰ يحتوي على خيارات التعديل والحذف (تظهر في نافذة منبثقة لتوفير المساحة).
- **🔍 تحسين البحث** – أصبح البحث يعمل بشكل موثوق على جميع المنصات (بما في ذلك تطبيق APK).
- **📱 تصميم متجاوب بالكامل** – واجهة مختلفة تماماً للحاسوب والهاتف، مع شريط سفلي مصغر على الحاسوب وتبويبات على الهاتف.
- **🔇 زر كتم الصوت** – إضافة زر mute بجانب شريط الصوت في كل من نسخة الحاسوب والهاتف.
- **🔄 خاصية التراجع عن الحذف (Undo)** – عند حذف إذاعة، تظهر رسالة "تم الحذف. تراجع؟" مع مؤقت 5 ثوانٍ لاستعادة الإذاعة المحذوفة.
- **📤 مشاركة الإذاعة والتطبيق** – إمكانية مشاركة رابط الإذاعة الحالية أو رابط مستودع GitHub للمشروع.

### 🐛 الإصلاحات

- إصلاح مشكلة **البحث** التي كانت لا تعمل في نسخة APK السابقة.
- إصلاح تداخل الأزرار مع النص في قائمة الإذاعات.
- تحسين استجابة التمرير الأفقي لتغيير الإذاعة (أعلى الشاشة) والصوت (أسفل الشاشة).
- إصلاح اختفاء صور الإذاعات في بعض الحالات.
- إصلاح مشكلة الشاشة الزرقاء في تطبيق Linux AppImage.

## 🛠️ التقنيات المستخدمة

- **HTML5 / CSS3** – البنية الأساسية، تصميم Glassmorphism، Flexbox/Grid
- **JavaScript (ES6+)** – منطق التطبيق، Web Audio API، Canvas API
- **LocalStorage API** – حفظ قائمة الإذاعات والمفضلة
- **Media Session API** – التحكم من شاشة القفل (قيد التطوير)
- **Service Workers** – للتشغيل دون اتصال وتحسين الأداء
- **Web App Manifest** – لتثبيت التطبيق كـ PWA
- **Capacitor** – لبناء تطبيق Android
- **Electron** – لبناء تطبيق Linux (AppImage/DEB)
- **Font Awesome 6** – أيقونات احترافية
- **Google Fonts (Tajawal)** – خط عربي حديث

## 🚀 كيفية الاستخدام

### 🌐 **عبر الإنترنت (PWA)**
1. انتقل إلى [الموقع على GitHub Pages](https://salehgnutux.github.io/GT_QURANRADIO/app/)
2. اضغط على زر "تثبيت التطبيق" في شريط العنوان (أو من القائمة)
3. سيظهر التطبيق في شاشة رئيسية جهازك كتطبيق مستقل

### 📱 **تطبيق أندرويد (APK)**
1. حمل ملف APK من الرابط أعلاه
2. اسمح بتثبيت التطبيقات من مصادر غير معروفة (في إعدادات الهاتف)
3. ثبّت التطبيق وابدأ الاستماع

### 🐧 **تطبيق لينكس (AppImage/DEB)**
```bash
# تشغيل AppImage
chmod +x QuranRadio-2.1.0-linux.AppImage
./QuranRadio-2.1.0-linux.AppImage

# تثبيت DEB
sudo dpkg -i QuranRadio-2.1.0-linux.deb
sudo apt-get install -f
💻 محلياً (للتطوير)
bash
git clone https://github.com/SalehGNUTUX/GT_QURANRADIO.git
cd GT_QURANRADIO
python -m http.server 8000   # أو npx serve
📋 متطلبات النظام
المنصة	المتطلبات
🤖 Android	الإصدار 8.0 (API 26) أو أحدث
🐧 Linux	أي توزيعة حديثة (Ubuntu 20.04+, Fedora 34+, Arch)
🌐 PWA / Web	متصفح حديث (Chrome, Firefox, Edge, Safari)
🔧 أوامر البناء من المصدر
bash
# استنساخ المستودع
git clone https://github.com/SalehGNUTUX/GT_QURANRADIO.git
cd GT_QURANRADIO

# إعطاء صلاحيات التنفيذ
chmod +x scripts/*.sh

# بناء Android APK
export ANDROID_HOME=$HOME/Android/Sdk
./scripts/build-android.sh

# بناء Linux (AppImage + DEB)
./scripts/build-linux.sh

# بناء الكل
./scripts/build-all.sh
المتطلبات:

Node.js ≥ 18

JDK 17 (لـ Android)

Android SDK (لـ Android)

Python 3 + Pillow (للأيقونات)
```
