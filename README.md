# مشغل الإذاعات القرآنية

تطبيق ويب تقدمي (PWA) متكامل للاستماع إلى الإذاعات القرآنية من مختلف أنحاء العالم الإسلامي، مع تصميم عصري وواجهة مستخدم سلسة.

## ✨ المميزات

- 🎵 **مشغل صوتي متطور** مع ذبذبات صوتية دائرية حول صورة الإذاعة
- 📱 **تطبيق PWA كامل** – يمكن تثبيته على الهاتف أو الحاسوب ويعمل كتطبيق مستقل
- 🎨 **واجهة مستخدم عصرية** مع تأثيرات زجاجية (Glassmorphism) وخلفيات متحركة
- 🖼️ **أيقونات متكاملة** بأحجام مختلفة (من 16×16 حتى 1024×1024) لجميع المنصات
- ➕ **إدارة كاملة للإذاعات** – إضافة، تعديل، حذف، واستيراد/تصدير البيانات
- 🔍 **تصفية الإذاعات** حسب الحالة (نشطة / جميع)
- 🌐 **دعم كامل للغة العربية** مع خط Tajawal الحديث
- 📱 **تصميم متجاوب (R)esponsive** يعمل بشكل مثالي على جميع الأجهزة
- 🔊 **تحكم كامل بالصوت** مع شريط تقدم، مؤقت، ومستوى صوت
- ⌨️ **اختصارات لوحة المفاتيح** (مفتاح المسافة للتشغيل/الإيقاف، الأسهم للتنقل)
- 🎯 **تحديث أوتوماتيكي لحالة الإذاعات**
- 💾 **تخزين محلي** (LocalStorage) لحفظ قائمة الإذاعات المفضلة
- 🚀 **Service Worker** متقدم للتشغيل دون اتصال (Offline-ready)

## 🛠️ التقنيات المستخدمة

- **HTML5** – البنية الأساسية للتطبيق
- **CSS3** – تصميم متقدم مع:
  - Flexbox & Grid
  - Glassmorphism (تأثيرات الزجاج)
  - Animations & Keyframes
  - Media Queries للتوافق مع جميع الأجهزة
- **JavaScript (ES6+)** – منطق التطبيق بالكامل مع:
  - Web Audio API (لذبذبات الصوت)
  - Canvas API (لرسم الموجات الدائرية)
  - LocalStorage API (لحفظ البيانات)
  - Media Session API (للتحكم من شاشة القفل)
- **Service Workers** – للتشغيل دون اتصال وتحسين الأداء
- **Web App Manifest** – لتثبيت التطبيق كـ PWA
- **Font Awesome 6** – أيقونات احترافية
- **Google Fonts (Tajawal)** – خط عربي حديث وجذاب

## 🚀 كيفية الاستخدام

### عبر الإنترنت (GitHub Pages)
1. انتقل إلى [الموقع على GitHub Pages](https://[اسم المستخدم].github.io/quran-radio)
2. اضغط على زر "تثبيت التطبيق" في شريط العنوان (أو من القائمة)
3. سيظهر التطبيق في شاشة رئيسية جهازك كتطبيق مستقل

### محلياً (للتطوير)
```bash
git clone https://github.com/SalehGNUTUX/GT_QURANRADIO.git
cd GT_QURANRADIO
# استخدم أي خادم محلي، مثلاً:
python -m http.server 8000
# أو
npx serve
```

### الميزات التفاعلية
- **تشغيل/إيقاف**: زر التشغيل المركزي أو مفتاح المسافة (Space)
- **التنقل بين الإذاعات**: أزرار "السابقة/التالية" أو مفاتيح الأسهم (← →)
- **تعديل الإذاعة**: انقر على زر التعديل في البطاقة الرئيسية أو في قائمة الإذاعات
- **استيراد/تصدير**: عبر القائمة الجانبية (أيقونة ☰ أعلى اليسار)

## 🎨 الأيقونات الجديدة

تم استخدام أيقونة المشروع الجديدة `GT-QURANRADIO-LOGO.png` في:
- أيقونة تبويب المتصفح (Favicon)
- أيقونة التطبيق على شاشة الهاتف (PWA)
- صورة احتياطية للإذاعات التي لا تتوفر على صورة
- شعار الموقع في رأس الصفحة

جميع الأيقونات متوفرة بأحجام مختلفة من 16×16 حتى 1024×1024 لتناسب جميع المنصات (iOS, Android, Windows, macOS, Linux).

## 🌟 مميزات PWA

- ✅ **تثبيت على الجهاز** – يعمل كتطبيق مستقل بدون شريط المتصفح
