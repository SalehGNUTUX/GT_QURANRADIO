## ✨ المميزات

### 🎵 **المشغل والمكتبة**
- مشغل صوتي متطور مع **ذبذبات صوتية دائرية** حول صورة الإذاعة أثناء التشغيل.
- قائمة تضم أكثر من **36 إذاعة قرآنية** جاهزة (يمكن إضافة المزيد).
- **إدارة كاملة للإذاعات**: إضافة، تعديل، حذف، استيراد/تصدير البيانات.
- **المفضلة (⭐)**: إضافة الإذاعات إلى المفضلة، ورفعها تلقائياً إلى أعلى القائمة بنفس ترتيب الإضافة.

### 🎨 **الواجهة والتصميم**
- تصميم عصري بتأثيرات **زجاجية (Glassmorphism)** وخلفيات متحركة.
- **ثلاث علامات تبويب** (الآن – الإذاعات – الإعدادات) لسهولة التنقل.
- دعم كامل للغة العربية مع خط **Tajawal** الحديث.
- تصميم متجاوب (Responsive) يعمل بشكل مثالي على جميع الأجهزة (هواتف، أجهزة لوحية، حواسيب).

### 🔊 **التحكم والتشغيل**
- **التحكم بالإصبع**:
  - التمرير في الجزء العلوي من الشاشة → تغيير الإذاعة.
  - التمرير في الجزء السفلي من الشاشة → تغيير مستوى الصوت.
- شريط تقدم، مؤقت، وتحكم بمستوى الصوت.
- **اختصارات لوحة المفاتيح** (مسافة للتشغيل/الإيقاف، الأسهم للتنقل).

### 📱 **مميزات تطبيق الهاتف (APK)**
- يعمل كتطبيق مستقل بعد التثبيت.
- واجهة سريعة وسلسة.
- حفظ الإذاعات المفضلة محلياً.

## 🛠️ التقنيات المستخدمة

| التقنية | الاستخدام |
|----------|------------|
| **HTML5 / CSS3** | البنية الأساسية، تصميم Glassmorphism، Flexbox/Grid |
| **JavaScript (ES6+)** | منطق التطبيق، Web Audio API، Canvas API |
| **LocalStorage API** | حفظ قائمة الإذاعات والمفضلة |
| **Media Session API** | التحكم من شاشة القفل (قيد التطوير) |
| **Service Workers** | للتشغيل دون اتصال وتحسين الأداء |
| **Web App Manifest** | لتثبيت التطبيق كـ PWA |
| **Font Awesome 6** | أيقونات احترافية |
| **Google Fonts (Tajawal)** | خط عربي حديث |

## 🚀 كيفية الاستخدام

### 🌐 **عبر الإنترنت (PWA)**
1. انتقل إلى [الموقع على GitHub Pages](https://salehgnutux.github.io/GT_QURANRADIO/)
2. اضغط على زر "تثبيت التطبيق" في شريط العنوان
3. سيظهر التطبيق في شاشة رئيسية جهازك كتطبيق مستقل

### 📱 **تطبيق أندرويد (APK)**
1. حمل ملف APK من الرابط أعلاه
2. اسمح بتثبيت التطبيقات من مصادر غير معروفة (في إعدادات الهاتف)
3. ثبّت التطبيق وابدأ الاستماع

### 💻 **محلياً (للتطوير)**
```bash
git clone https://github.com/SalehGNUTUX/GT_QURANRADIO.git
cd GT_QURANRADIO
python -m http.server 8000   # أو npx serve
```

## 🚧 **القيود الحالية (إصدار 2.1)**

| الميزة | الحالة |
|--------|--------|
| التشغيل في الخلفية (Background Audio) | ❌ **لا يعمل حالياً** |

> ⚠️ **ملاحظة**: العمل في الخلفية وإشعارات شاشة الققل من الميزات المخطط لها في الإصدارات القادمة. حالياً، يستمر التطبيق بالعمل فقط عند فتحه على الشاشة.
